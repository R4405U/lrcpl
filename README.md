# lrc.pl
perl script to fetch lyrics from LRCLIB using LRCLIB API

Today:   11/01/2026 Sunday

# Requirements
Getopt::Long 
Image::ExifTool 
LWP::UserAgent 
LWP::Protocol:https 
JSON 
URI::Escape 

You can install all these using given below command

```bash
cpanm --installdeps .
```

or you can install using your native distro package manager
